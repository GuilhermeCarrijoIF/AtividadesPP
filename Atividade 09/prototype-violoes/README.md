# Prototype - Fábrica de Violões

Exemplo em Java usando o padrão Prototype com:

- clonagem superficial;
- clonagem profunda;
- cliente `Main` escolhendo um tipo de violão da fábrica.

## Executar

```bash
mvn compile
mvn exec:java -Dexec.mainClass="br.com.fabrica.violoes.Main"
```

Ou compile diretamente com `javac` se preferir.
